## Observações do projeto
### Willian Junior Custodio Firmo

Embora o projeto tenha sido inicialmente planejado para ser desenvolvido em dupla, enfrentamos problemas de organização e administração de tempo, tanto da minha parte quanto a dele. A integração do backend no frontend iria ser realizado nesse final de semana.

No entanto, meu colega não disponibilizou a parte dele no GitHub, o que impossibilitou o andamento do trabalho. Aguardei até segunda-feira, um dia antes da entrega, para tentar recuperar o código que minha dupla havia deixado em um dos notebooks da faculdade, porem o projeto não estava em nenhum deles.

Dessa forma, além de não receber uma versão minimamente executável para realizar testes e validações, também não tive acesso ao repositório atualizado. Consequentemente, concluí apenas o frontend, no qual eu estava encarregado, com a ressalva que o processamento de dados que deveriam ter vindo do backend, ficou no frontend.
